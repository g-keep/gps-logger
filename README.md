# GPS-LOGGER

Gps-Logger is a Python app that read NMEA Frame from the serial port and write the decoded frame into a json file.

## Installation

Use the package manager [pip] (https://pip.pypa.io/en/stable/) to install.



## Usage

python gps-logger.py
